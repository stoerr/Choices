About Group Choices

Purpose

This site can support a group making a decision between several choices by letting everyone give a rating to each choice, and summing up the result. No registration is required - after creating a poll you get a link you can send around e.g. per email to your peers. Everybody who has this link can make a vote and see the results of the poll.

Status

As you can see, this site is quite incomplete yet, but it is possible to create polls, make votes and inspect the results. If you care, you can have a look at the source at GitHub. Feedback is welcome.

Thanks

This site was kicked off in a hackathon event at T-Systems Multimedia Solutions. Stefan Scheidewig did a quick original HTML design with Bootstrap, the Scala code was mostly done by Hans-Peter Störr, and there are some contributions of Markus Böhme. The main idea was to have some fun with the amazingly productive Scala programming language. But using Google app engine this quickly became something actually usable, afterwards.
