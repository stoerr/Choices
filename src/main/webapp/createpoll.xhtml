Create a new poll

This site can support a group making a decision between several choices by letting everyone give a rating to each choice, and summing up the result. No registration is required - after creating a poll you get a link you can send around e.g. per email to your peers. Everybody who has this link can make a vote and see the results of the poll.

As you can see, this site is quite incomplete yet, but it is already possible to create polls, make votes and see the results.

CAUTION: Please don't use this for anything important, or where anybody could be more than mildly annoyed if anything goes wrong. This was done for fun; nobody got payed anything for any of that. So, there are absolutely no warranties. That said, feedback is welcome.
